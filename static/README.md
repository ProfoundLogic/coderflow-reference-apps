# static — plain HTML/CSS/JS (no backend)

The simplest reference app: hand-written `index.html`, `styles.css`, and `app.js`, served as static files. No framework, no build step, no backend — edit a file and refresh.

Serve it with any static server (in CoderFlow, e.g. `npx serve .` or `python -m http.server`), or copy the files into a backend's static folder.

**See:** [Stack reference → front ends](https://coderflow.ai/docs/environment-setup/web-applications/stack-reference).
